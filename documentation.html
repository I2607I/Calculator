SmartCalc

SmartCalc v.1.0

 

SmartCalc - это калькулятор, с возможностью построения графиков функций. Для вычисления выражения используются два стека, которые определяют порядок операций находящихся в выражении.

 

Доступные функции:

Кнопки от "0" до "9";
Кнопка "." - для вычисления дробных выражений;
Кнопки "+" "-" "*" "/" - стандарные арифметические операции;
Кнопка "mod" - для получения остатка от деления на число (пример использования - 5mod10);
Кнопка "^" - для возведения числа в степень (пример использования - 10^10);
Кнопка "sqrt" - для получения корня числа (пример использования - sqrt(16));
Кнопки "(" ")" - для определения порядка расчетов (пример использования - (3+2)-(2*2));
Кнопка "cos" - для получения косинуса числа (пример использования - cos(2));
Кнопка "sin" - для получения синуса числа (пример использования - sin(10));
Кнопка "tan" - для получения тангенса числа (пример использования - sin(10));
Кнопка "acos" - для получения арккосинуса числа (пример использования - acos(0.5));
Кнопка "asin" - для получения арксинуса числа (пример использования - asin(-0.5));
Кнопка "atan" - для получения арктанса числа (пример использования - atan(0.25));
Кнопка "ln" - для получения логарифма числа (пример использования - ln(100));
Кнопка "log" - для получения десятичного логарифма числа (пример использования - log(10));
Кнопка "=" - для получения результата введенного выражения;
Кнопка "AC" - для очистки строки от выражения.

Для построения графика используется кнопка "GR".

 

Работа в кредитном калькуляторе. Кнопка Кнопка "CRED"  

Для того, чтобы сделать расчет в кредитном калькуляторе необходимо ввести значения: сумма кредита, процентная ставка и период.

Кнопка "C" очищает поле ввода.
